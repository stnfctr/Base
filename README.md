# Base Ekosistemi Topluluk Kütüphanesi 🛠️

Bu depo, Base ekosistemi üzerinde geliştirme yapan geliştiriciler için oluşturulmuş, topluluk odaklı bir kaynak merkezidir. Misyonumuz, açık kaynaklı araçlar, entegrasyon kılavuzları ve pratik örnekler sağlayarak Base üzerinde gerçek kullanım odaklı projelerin geliştirilmesine katkıda bulunmaktır.

---

## ⚠️ Önemli

**Yasal Uyarı:** Bu depo, resmi Base veya Coinbase kuruluşlarıyla bağlantılı değildir, onlar tarafından yönetilmemekte, yetkilendirilmemekte veya onaylanmamaktadır. Geliştirici ekosistemini desteklemek amacıyla tasarlanmış bağımsız, topluluk tarafından yönetilen bir girişimdir.

---

## 🚀 Mevcut Modüller

Bu kütüphanede yer alan güncel projeler ve dokümantasyon:

### ⚡ Base Geliştirici Kılavuzu

Base ağı üzerinde akıllı sözleşme dağıtımı, ödeme sistemleri ve ölçeklenebilir dApp mimarileri hakkında kapsamlı bir teknik kılavuz.

- **Durum:** Aktif Geliştirme Aşamasında (Base Mainnet & Testnet)
- **Kapsam:**  
  - Cüzdan entegrasyonları  
  - x402 ödeme mimarisi (pay-per-request)  
  - USDC ile mikro ödeme sistemleri  
  - API monetization (Pay-per-API yapıları)

---

## 🤝 Nasıl Katkıda Bulunabilirsiniz?

Bu kütüphane "Birlikte İnşa Etme" vizyonu üzerine kurulmuştur. Çeşitli yollarla katkıda bulunabilirsiniz:

- **Kod Örnekleri:** Base üzerinde çalışan yardımcı scriptler, SDK örnekleri veya template projeler paylaşın  
- **Dokümantasyon:** Mevcut kılavuzları geliştirin, eksikleri tamamlayın veya yeni kullanım senaryoları ekleyin  
- **Yeni Modüller:** Base ile diğer ağlar (Ethereum, Arbitrum, Scroll vb.) arasında köprüleme, veri akışı veya ödeme sistemleri üzerine içerik ekleyin  

---

## 🛠️ Teknoloji Altyapısı ve Araçlar

Bu kütüphanedeki projeler genellikle şunları kullanır:

- **Altyapı:** Akıllı sözleşmeler, onchain ödeme sistemleri  
- **Ağlar:** Base, Ethereum, L2 ekosistemi  
- **Geliştirme:** Solidity, TypeScript, Node.js, Hardhat, Ethers.js, Viem
